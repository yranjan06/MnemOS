// how it works //
four steps. one container.
01
design workflow
Drag nodes onto the canvas. Connect them. Configure each node's task, LLM, and parameters.
02
generate code
Hit Generate. The backend compiles your graph into a Python workflow.py — inspect it in the preview pane.
03
run in docker
Hit Run. The agent executes inside the VM container with full browser access. Watch logs live.
04
memory persists
Remember nodes write to HydraDB. Every future run recalls past context. The agent compounds knowledge.
